# Third-Party Notices

This repository distributes third-party open-source dependencies through `package.json` and `package-lock.json`.

- Primary dependency inventory: `package.json`
- Resolved lockfile inventory: `package-lock.json`
- CI supply-chain audit: `npm run audit:ci-supply-chain`
- Security scan gate: `npm run security:scan`

Dependency-specific license texts remain governed by the corresponding upstream packages.
