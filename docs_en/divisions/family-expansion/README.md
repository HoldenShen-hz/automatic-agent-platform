# Family Expansion Reports

This directory records the governance closure status when the v3.2 Family pilot expands to more canonical divisions.

Current layers:

- `engineering-expansion.md`
- `knowledge-research-expansion.md`
- `enterprise-ops-expansion.md`
- `gtm-content-expansion.md`

Maintenance rules:

- Only records expansion plans aligned with `config/quality/division-catalog.json`.
- Does not redefine runtime family here; only governance grouping expansion strategy to canonical divisions.
- Any new family report must indicate `pilot source`, `target divisions`, `blocked risks`, `required evidence`.
