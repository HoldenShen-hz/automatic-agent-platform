# Migration

The `migration/` directory answers only two questions: how to migrate and what to migrate.

## File Order

1. [00-migration-guideline.md](./00-migration-guideline.md)
   Migration principles, phases, absorption methods, and judgment criteria.
2. [01-migration-scope.md](./01-migration-scope.md)
   Official migration boundaries and exclusions for the new platform.

## Usage Principles

- First apply the guideline to judge, then execute according to scope.
- When conflicts arise with the platform skeleton, [architecture/00-platform-architecture.md](../architecture/00-platform-architecture.md) takes precedence.
