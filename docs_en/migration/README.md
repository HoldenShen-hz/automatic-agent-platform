# Migration

`migration/` only answers two questions: how to migrate and what to migrate.

## File Order

1. [00-migration-guideline.md](./00-migration-guideline.md)
   Migration principles, levels, absorption methods, and judgment criteria.
2. [01-migration-scope.md](./01-migration-scope.md)
   New platform formal migration boundaries and exclusions.

## Usage Principles

- First judge according to guidelines, then execute according to scope.
- When conflicts arise with the platform skeleton, [architecture/00-platform-architecture.md](../architecture/00-platform-architecture.md) takes precedence.
