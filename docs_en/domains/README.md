## Domain Document Structure

Each business domain directory typically contains:

- `domain.yaml` - Domain definitions and configurations
- `workflows/` - Domain workflows
- `roles/` - Domain role definitions
- `plugins/` - Domain plugin configurations

## Usage Rules

- Business domains are capability boundaries within the platform.
- Each business domain may have its own workflows, roles, and tools.
- Domain onboarding and registration follows the domain registry as the source of truth.
